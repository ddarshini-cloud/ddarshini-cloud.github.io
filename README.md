# Divya Darshini M - Portfolio

A professional portfolio website showcasing my education, skills, projects, and certifications as a Computer Science and Engineering student.

## Features
- Responsive design that works on all devices
- Clean, modern interface with professional color scheme
- Smooth scrolling navigation
- Organized sections for easy information access
- Interactive elements and hover effects

## Live Demo
[View Live Portfolio](https://ddarshini-cloud.github.io/portfolio)

## Technologies Used
- HTML5
- CSS3 (with CSS Variables for theming)
- JavaScript (ES6)
- Font Awesome Icons
- Google Fonts (Poppins & Roboto)

## Sections Included
1. **Home** - Introduction and hero section
2. **About** - Personal profile and background
3. **Education** - Academic timeline
4. **Skills** - Technical and professional skills
5. **Projects** - Key projects and achievements
6. **Certifications** - Professional certifications
7. **Contact** - Contact information and hobbies

## Setup Instructions
1. Fork or clone this repository
2. Open `index.html` in your browser
3. Customize the content to match your information
4. Deploy to GitHub Pages for live hosting

## Deployment to GitHub Pages
1. Create a new repository on GitHub
2. Upload all files (index.html, styles.css, script.js)
3. Go to Repository Settings > Pages
4. Select "main" branch as source
5. Your site will be published at `https://yourusername.github.io/repository-name`

## Customization
- Update personal information in `index.html`
- Modify colors in `:root` variables in `styles.css`
- Add or remove sections as needed
- Replace placeholder image with your photo
